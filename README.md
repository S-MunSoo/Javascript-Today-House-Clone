#### 📌 Local에서 쉽게 서버 띄워보기(Live Server)

1. [Live Server](https://www.npmjs.com/package/live-server) 다운받기

   ```
     npm install -g live-server
   ```

2. 실행하기

   ```
     live-server 실행할_파일들이_있는_폴더명
   ```

#### 📌 Local에서 데이터베이스 서버 띄워보기(Json Server)

1. json-server 설치하기

   ```
     npm install json-server
   ```

2. json-server 실행하기

   ```
     npm run serve
   ```
